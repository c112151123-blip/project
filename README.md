# 高科失物招領系統

一個為學校設計的失物招領管理系統，幫助學生和教職員快速找回丟失的物品。

##  功能特性

### 核心功能

#### 用戶管理
- **三種用戶角色**：
  - **管理員**：負責上傳和管理失物及拾物
  - **學生**：可搜尋失物和拾物，提交認領申請
  - **教職員**：同學生，具有相同的功能
- 基於學號或編號（ID）的登錄和身份識別
- 個人信息管理和儀表板

#### 管理員功能
- **失物管理**：上傳、編輯、刪除失物記錄
- **拾物管理**：上傳、編輯、刪除拾物記錄
- 物品分類管理
- 詳細的物品信息錄入（名稱、描述、地點、日期、聯絡方式）

#### 一般用戶功能
- **搜尋失物**：按類別或關鍵詞搜尋丟失的物品
- **搜尋拾物**：查看系統中已上傳的拾物
- **提交認領申請**：對相符的物品提交認領申請
- **跟蹤申請狀態**：查看認領申請的進展

#### 系統功能
- **物品匹配**：根據分類自動推薦相似物品
- **認領流程**：
  - 用戶提交認領申請
  - 管理員審核並批准/拒絕
  - 物品狀態自動更新
- **儀表板**：
  - 管理員可查看上傳的失物和拾物
  - 用戶可查看自己的認領申請

## 快速開始

### 前置要求
- Python 3.8 或更高版本
- pip 包管理器
- 虛擬環境（推薦）

### 安裝步驟

1. **創建虛擬環境**
   ```bash
   python -m venv .venv
   ```

2. **激活虛擬環境**
   
   **Windows:**
   ```bash
   .venv\Scripts\activate
   ```
   
   **macOS/Linux:**
   ```bash
   source .venv/bin/activate
   ```

3. **安裝依賴**
   ```bash
   pip install -r requirements.txt
   ```

4. **運行應用**
   ```bash
   python run.py
   ```

5. **訪問應用**
   
   打開瀏覽器訪問：http://localhost:5000

## 📱 使用指南

### 1. 註冊賬戶
- 點擊"註冊"按鈕
- 填寫用戶信息（用戶名、電郵、真實姓名等）
- 選擇身份（學生或教職員）
- 提交註冊

### 2. 登入系統
- 使用註冊的用戶名和密碼登入
- 進入個人儀表板

### 3. 報告失物
- 點擊"報告失物"按鈕
- 填寫詳細信息：
  - 物品名稱
  - 物品類別
  - 丟失地點
  - 丟失日期和時間
  - 詳細描述（特別是識別特徵）
  - 聯絡電話（可選）
- 提交報告

### 4. 報告拾物
- 點擊"報告拾物"按鈕
- 填寫詳細信息：
  - 物品名稱
  - 物品類別
  - 拾獲地點
  - 拾獲日期和時間
  - 詳細描述
  - 聯絡電話（可選）
- 提交報告

### 5. 搜索物品
- 瀏覽"失物尋找"或"拾物查詢"
- 使用搜索框或過濾器查找
- 點擊"查看詳情"了解更多信息

### 6. 認領物品
- 在失物詳情頁面，查看匹配的拾物
- 如果發現您的物品，點擊"認領申請"
- 提供識別信息以證明物品所有權
- 等待拾物人確認

### 7. 管理您的報告
- 在"我的儀表板"查看所有報告
- 可以編輯或刪除您的報告
- 追蹤認領申請的狀態

## 🗄️ 數據庫模型

### User（用戶）
- 學號/編號（唯一，用於登入）
- 密碼、電郵
- 真實姓名、身份（學生/教職員/管理員）
- 建立時間

### LostItem（失物）
- 物品名稱、描述、類別
- 丟失地點、丟失日期
- 報告人（通常為管理員）、狀態
- 聯絡電話

### FoundItem（拾物）
- 物品名稱、描述、類別
- 拾獲地點、拾獲日期
- 拾物人（通常為管理員）、狀態
- 聯絡電話

### Claim（認領記錄）
- 失物ID、拾物ID、申請人
- 認領狀態（待批准、已批准、已拒絕、已歸還）
- 申請說明、申請時間

##  常見問題

### Q1：如何確認找到的物品確實是我的？
A：提交認領申請時，需要提供物品的詳細識別信息（顏色、標記、序列號等），拾物人會根據這些信息確認。

### Q2：失物多久沒有被認領會被如何處理？
A：建議學校定期檢查系統中的拾物報告，設定合理的保留期限（如30天或60天），過期未認領的物品可以進行捐贈或回收。

### Q3：如何聯絡拾物人？
A：每個拾物報告都會顯示拾物人的聯絡電話。失主可以直接聯絡拾物人。

### Q4：系統如何防止虛假認領？
A：要求申請人在認領表單中提供詳細的物品識別信息，由拾物人進行人工確認。
