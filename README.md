# 高科失物招領系統

一個為學校設計的失物招領管理系統，幫助學生和教職員快速找回丟失的物品。

##  功能特性

### 核心功能

#### 用戶管理
- **三種用戶角色**：
  - **學生**：報告失物、搜尋拾物、提交認領申請
  - **教職員**：報告失物、搜尋拾物、提交認領申請
  - **管理員**：上傳拾物、管理系統、批准認領申請
- 基於學號或編號（ID）的登錄和身份識別
- 個人信息管理和儀表板

#### 一般用戶功能
- **報告失物**：輸入丟失物品的詳細信息（名稱、描述、地點、日期等）
- **搜尋拾物**：瀏覽系統中已上傳的拾物記錄
- **提交認領申請**：對相符的拾物提交認領請求
- **管理失物**：編輯或刪除自己報告的失物
- **跟蹤申請**：查看認領申請的審核狀態

#### 管理員功能
- **上傳拾物**：添加學校內找到的物品到系統
- **管理拾物**：編輯或刪除拾物記錄
- **審核認領**：批准或拒絕用戶的認領申請
- **專門登入入口**：增強的安全性和系統保護

#### 系統功能
- **物品匹配**：根據分類自動推薦相似物品
- **認領流程**：
  - 用戶提交認領申請
  - 管理員審核並批准/拒絕
  - 物品狀態自動更新
- **儀表板**：
  - 一般用戶查看自己報告的失物和認領申請
  - 管理員查看系統內的所有拾物

## 快速開始

### 前置要求
- Python 3.8 或更高版本
- pip 包管理器
- 虛擬環境（推薦）

### 安裝步驟

1. **創建虛擬環境**
   ```bash
   python -m venv .venv
   ```

2. **激活虛擬環境**
   
   **Windows:**
   ```bash
   .venv\Scripts\activate
   ```
   
   **macOS/Linux:**
   ```bash
   source .venv/bin/activate
   ```

3. **安裝依賴**
   ```bash
   pip install -r requirements.txt
   ```

4. **運行應用**
   ```bash
   python run.py
   ```

5. **訪問應用**
   
   打開瀏覽器訪問：http://localhost:5000

## 📱 使用指南

### 普通用戶（學生/教職員）工作流程

#### 1. 註冊賬戶
- 點擊"註冊"按鈕
- 填寫用戶信息：
  - 學號/編號：用於登錄的唯一標識
  - 電郵地址
  - 真實姓名
  - 身份：選擇「學生」或「教職員」
  - 密碼
- 點擊"立即註冊"

#### 2. 登入系統
- 在首頁點擊"登入"
- 輸入學號/編號和密碼
- 進入個人儀表板

#### 3. 報告遺失的物品
- 點擊導航菜單中的"報告失物"
- 填寫詳細信息：
  - 物品名稱（例：黑色iPhone）
  - 物品類別（電話、錢包、鑰匙等）
  - 丟失地點
  - 丟失日期
  - 詳細描述（包括識別特徵，如顏色、損傷等）
  - 聯絡電話（可選）
- 點擊"提交報告"

#### 4. 搜尋和查看拾物
- 點擊首頁的"拾物查詢"
- 瀏覽系統中管理員上傳的拾物
- 使用搜索框或類別過濾尋找相符物品
- 點擊"查看詳情"了解更多信息

#### 5. 認領找到的物品
- 找到相符的拾物後，點擊"認領申請"
- 提供識別信息（描述該物品的特徵以證明所有權）
- 點擊"提交申請"
- 等待管理員審核和批准

#### 6. 查看申請狀態
- 在"我的儀表板"查看"我的認領申請"
- 可以看到申請的當前狀態

#### 7. 管理失物報告
- 在儀表板查看和編輯自己報告的失物
- 可點擊"詳情"查看認領情況

---

### 管理員工作流程

#### 1. 管理員登入
- 點擊首頁的"管理員登入"（紅色按鈕）
- 輸入管理員編號和密碼
- 進入管理員儀表板

#### 2. 上傳拾物
- 在導航菜單中點擊"上傳拾物"
- 填寫物品信息並提交

#### 3. 管理拾物記錄
- 在管理員儀表板查看和編輯拾物
- 審核認領申請

## 🗄️ 數據庫模型

### User（用戶）
- 學號/編號（唯一，用於登入）
- 密碼、電郵
- 真實姓名、身份（學生/教職員/管理員）
- 建立時間

### LostItem（失物）
- 物品名稱、描述、類別
- 丟失地點、丟失日期
- 報告人（普通用戶報告自己的失物）、狀態
- 聯絡電話

### FoundItem（拾物）
- 物品名稱、描述、類別
- 拾獲地點、拾獲日期
- 拾物人（通常為管理員）、狀態
- 聯絡電話

### Claim（認領記錄）
- 失物ID、拾物ID、申請人
- 認領狀態（待批准、已批准、已拒絕、已歸還）
- 申請說明、申請時間

##  安全特性

- **密碼安全**：所有密碼使用加密存儲
- **用戶認證**：基於會話的安全認證機制
- **權限管理**：
  - 只有報告者可編輯/刪除自己報告的失物
  - 只有管理員可上傳和編輯拾物
  - 只有授權的管理員可通過管理員入口登入
- **安全隔離**：普通用戶和管理員有各自獨立的登入入口
- **CSRF 保護**：防止跨站請求偽造

##  常見問題

### Q1：如何確認找到的物品確實是我的？
A：提交認領申請時，需要提供物品的詳細識別信息（顏色、標記、序列號等），拾物人會根據這些信息確認。

### Q2：失物多久沒有被認領會被如何處理？
A：建議學校定期檢查系統中的拾物報告，設定合理的保留期限（如30天或60天），過期未認領的物品可以進行捐贈或回收。

### Q3：如何聯絡拾物人？
A：每個拾物報告都會顯示拾物人的聯絡電話。失主可以直接聯絡拾物人。

### Q4：系統如何防止虛假認領？
A：要求申請人在認領表單中提供詳細的物品識別信息，由拾物人進行人工確認。
