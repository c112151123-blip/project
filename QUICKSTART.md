# 快速開始指南

## 環境準備

### Windows 系統
1. **打開命令提示符或 PowerShell**
   
2. **創建虛擬環境**
   ```cmd
   python -m venv .venv
   ```

3. **激活虛擬環境**
   ```cmd
   .venv\Scripts\activate
   ```

4. **安裝依賴包**
   ```cmd
   pip install -r requirements.txt
   ```

5. **啟動應用**
   ```cmd
   python run.py
   ```

6. **打開瀏覽器訪問**
   - 地址：http://localhost:5000
   - 首次訪問自動創建數據庫

---

### macOS / Linux 系統
1. **打開終端**

2. **創建虛擬環境**
   ```bash
   python3 -m venv .venv
   ```

3. **激活虛擬環境**
   ```bash
   source .venv/bin/activate
   ```

4. **安裝依賴包**
   ```bash
   pip install -r requirements.txt
   ```

5. **啟動應用**
   ```bash
   python run.py
   ```

6. **打開瀏覽器訪問**
   - 地址：http://localhost:5000

## 首次使用

### 步驟 1：創建測試賬戶（學生）
1. 點擊"註冊"按鈕
2. 填寫表單：
   - 用戶名：`test_student`（所有人都能看到的名字）
   - 電郵：`test@school.com`
   - 真實姓名：`測試學生`
   - 身份：選擇「學生」
   - 學號：`STU20260001`（身份識別碼）
   - 密碼：`123456`
3. 點擊"立即註冊"

### 步驟 2：創建教職員賬戶（可選）
1. 點擊"註冊"按鈕
2. 填寫表單：
   - 用戶名：`test_staff`
   - 電郵：`staff@school.com`
   - 真實姓名：`測試教職員`
   - 身份：選擇「教職員」
   - 教職員編號：`STAFF2026001`（身份識別碼）
   - 密碼：`123456`
3. 點擊"立即註冊"

### 步驟 3：登入系統
1. 使用用戶名和密碼登入
2. 進入儀表板

### 步驟 4：報告失物（示例）
1. 點擊"報告失物"按鈕
2. 填寫表單：
   - 物品名稱：`黑色iPhone 13`
   - 類別：`電話`
   - 丟失地點：`圖書館三樓`
   - 丟失日期：選擇今天或昨天
   - 詳細描述：`黑色 iPhone 13，屏幕有貼膜，背面有一道小劃痕`
   - 聯絡電話：`13800138000`
3. 點擊"提交報告"

### 步驟 5：報告拾物（示例）
1. 使用教職員或另一個學生賬戶
2. 點擊"報告拾物"按鈕
3. 填寫與上述相似的物品信息
4. 點擊"提交報告"

### 步驟 6：查看匹配結果
1. 返回失物詳情頁面
2. 系統會自動展示匹配的拾物
3. 提交認領申請

## 數據庫位置

首次運行後，數據庫文件會自動生成在：
```
project/lost_found.db
```

**清除所有數據**（重新開始）：
1. 停止應用（按 Ctrl+C）
2. 刪除 `lost_found.db` 文件
3. 重新啟動應用

## 常見問題解決

### 問題 1：找不到 Flask 模塊
**解決方案**：確保虛擬環境已激活
```bash
# 檢查是否激活（命令提示符前應該有 (.venv)）
pip list
```

### 問題 2：端口 5000 已被使用
**解決方案**：修改啟動方式
```bash
python run.py  # 會自動嘗試其他端口
```

### 問題 3：中文顯示亂碼
**解決方案**：確保使用支持 UTF-8 的終端

## 進階功能

### 了解用戶識別系統
- 每個學生必須提供學號（作為唯一識別碼）
- 每個教職員必須提供編號（作為唯一識別碼）
- 用戶名是所有人都能看到的公開名字
- 真實姓名不會被公開顯示

### 添加管理員賬戶
編輯 `app/routes.py` 中的 `register()` 函數，在創建用戶時設置：
```python
role = 'admin'  # 創建管理員賬戶
```

### 修改物品分類
編輯 `app/routes.py`，查找 `categories` 列表並修改。

### 自定義樣式
編輯 `app/static/css/style.css` 修改外觀。

## 部署到生產環境

當準備部署到生產環境時：

1. **安裝生產服務器**
   ```bash
   pip install gunicorn
   ```

2. **更新配置**
   - 修改 `.env` 中的 `FLASK_ENV=production`
   - 生成強的 `SECRET_KEY`

3. **運行應用**
   ```bash
   gunicorn -w 4 -b 0.0.0.0:5000 run:app
   ```
